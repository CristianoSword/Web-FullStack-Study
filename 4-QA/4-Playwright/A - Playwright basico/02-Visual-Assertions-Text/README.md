# 02 - Visual Assertions Text

Projeto focado em validações de texto, labels e feedback visual usando os matchers do Playwright.

## O que tem aqui

- Formulário local com validação client-side.
- Dados esperados centralizados em `src/models/ui-copy.js`.
- Teste cobrindo visibilidade, placeholders e mensagens de erro/sucesso.

## Como rodar

```bash
npm install
npx playwright test
```
