# 05 - Auto Waiting Locators

Projeto focado em demonstrar o auto-waiting nativo do Playwright.

## O que tem aqui

- App local com conteúdo liberado de forma assíncrona.
- Estados esperados centralizados em `src/models/loading-states.js`.
- Teste sem `waitForTimeout`, usando apenas locators e assertions.

## Como rodar

```bash
npm install
npx playwright test
```
