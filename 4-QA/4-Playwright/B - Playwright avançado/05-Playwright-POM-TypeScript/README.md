# 05 - Playwright POM TypeScript

Projeto avançado com framework de testes baseado em Page Object Model com TypeScript.

## O que tem aqui

- `playwright.config.ts` e `tsconfig.json`.
- Page Objects em `src/pages`.
- Modelo tipado para o usuário em `src/models/study-user.ts`.
- Fluxo de login local exercitado por `tests/login-flow.spec.ts`.

## Como rodar

```bash
npm install
npx playwright test
```
