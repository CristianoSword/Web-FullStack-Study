# 05-JUnit-Maven-Gradle-Config

Laboratorio basico de **JUnit 5** focado em configurar a suite tanto com **Maven** quanto com **Gradle**.

## O que este projeto cobre

- `pom.xml` com Surefire
- `build.gradle` com `useJUnitPlatform()`
- classe de servico simples
- testes executaveis nos dois build tools

## Como estudar

```bash
mvn test
gradle test
```
