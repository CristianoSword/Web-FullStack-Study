# 01-Cypress-GUI-Installation

Laboratorio basico de **Cypress** focado em instalacao, abertura do painel grafico e primeiro spec local.

## O que este projeto cobre

- dependencia `cypress`
- scripts `cypress:open` e `cypress:run`
- configuracao basica em `cypress.config.js`
- primeira pagina local para smoke da GUI

## Como estudar

```bash
npm install
npm run serve
npm run cypress:open
```
