# 01-Complete-E2E-Authentication-Flow

Laboratorio avancado de **Cypress** para fluxo E2E completo de cadastro, login e redirecionamento.

## O que este projeto cobre

- cadastro de usuario em pagina local
- login com credenciais persistidas em `localStorage`
- redirecionamento para dashboard autenticada
- assercoes de navegacao e estado final

## Como estudar

```bash
npm install
npm run serve
npm run cypress:run
```
