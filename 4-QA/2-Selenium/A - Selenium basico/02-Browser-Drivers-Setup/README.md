# 02-Browser-Drivers-Setup

Laboratorio basico de **Selenium** para configurar Chrome, Firefox e Edge drivers em um projeto Node.

## O que este projeto cobre

- dependencia `selenium-webdriver`
- fabrica de browsers com Chrome, Firefox e Edge
- configuracao de paths de driver por ambiente
- smoke script abrindo pagina local

## Como estudar

```bash
npm install
set BROWSER=chrome
set CHROME_DRIVER_PATH=C:\drivers\chromedriver.exe
npm start
```
