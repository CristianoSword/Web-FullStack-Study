# 05-Selenium-Framework-JUnit-Asserts

Laboratorio avancado de **Selenium com JUnit 5** em um projeto Maven Java.

## O que este projeto cobre

- `pom.xml` com JUnit Jupiter e Selenium Java
- classe de teste com `Assertions`
- fixture para abrir pagina local em `src/test/resources`
- exemplo de suite orientada a framework

## Como estudar

```bash
mvn test
```
