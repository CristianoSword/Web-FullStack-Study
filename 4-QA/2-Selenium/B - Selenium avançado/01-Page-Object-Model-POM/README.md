# 01-Page-Object-Model-POM

Laboratorio avancado de **Selenium** usando **Page Object Model** para organizar o fluxo de login e dashboard.

## O que este projeto cobre

- `BasePage`, `LoginPage` e `DashboardPage`
- separacao entre localizadores e fluxo de teste
- pagina local para autenticacao
- script principal usando os page objects

## Como estudar

```bash
npm install
npm start
```
