# 03-React-Component-Testing

Laboratorio avancado de **Jest** com **Testing Library** para testes de componente React sem depender de backend.

## O que este projeto cobre

- ambiente `jsdom`
- `@testing-library/react` e `@testing-library/jest-dom`
- componente com estado e interacao do usuario
- assercoes acessiveis por papel e texto

## Estrutura

- `src/components/`: componente de progresso e cartao de trilha
- `tests/`: interacao, renderizacao e mudanca de estado
- `jest.config.cjs`: ambiente de DOM para os testes

## Como executar

```bash
npm install
npm test
```
