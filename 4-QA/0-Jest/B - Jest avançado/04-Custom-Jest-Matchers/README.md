# 04-Custom-Jest-Matchers

Laboratorio avancado de **Jest** para criar matchers customizados voltados a cenarios de estudo e progresso.

## O que este projeto cobre

- `expect.extend`
- mensagens personalizadas de falha
- matchers semanticos para progresso e identificadores
- testes usando os matchers novos

## Como executar

```bash
npm install
npm test
```
