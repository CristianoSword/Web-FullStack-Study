# 05-Performance-Benchmark-Tests

Laboratorio avancado de **Jest** para comparar tempo de execucao de estrategias simples dentro da suite.

## O que este projeto cobre

- uso de `performance.now()`
- comparacao entre algoritmos
- assercoes de limite de tempo razoavel
- relatorio simples de benchmark

## Como executar

```bash
npm install
npm test
```
