# PubSub Playbook

1. Execute `powershell -ExecutionPolicy Bypass -File .\scripts\run-infra.ps1`.
2. Rode `npm install`.
3. Inicie a API com `powershell -ExecutionPolicy Bypass -File .\scripts\start-api.ps1`.
4. Publique um broadcast com `powershell -ExecutionPolicy Bypass -File .\scripts\publish-broadcast.ps1`.
5. Consulte `powershell -ExecutionPolicy Bypass -File .\scripts\list-events.ps1`.
6. Consulte `powershell -ExecutionPolicy Bypass -File .\scripts\list-subscribers.ps1`.
7. Consulte `powershell -ExecutionPolicy Bypass -File .\scripts\list-deliveries.ps1`.
