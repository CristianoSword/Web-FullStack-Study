# Direct Routing Playbook

1. Execute `powershell -ExecutionPolicy Bypass -File .\scripts\run-infra.ps1`.
2. Rode `npm install`.
3. Inicie a API com `powershell -ExecutionPolicy Bypass -File .\scripts\start-api.ps1`.
4. Publique um erro com `powershell -ExecutionPolicy Bypass -File .\scripts\publish-error-log.ps1`.
5. Publique um warning com `powershell -ExecutionPolicy Bypass -File .\scripts\publish-warning-log.ps1`.
6. Consulte `powershell -ExecutionPolicy Bypass -File .\scripts\list-routes.ps1`.
7. Consulte `powershell -ExecutionPolicy Bypass -File .\scripts\list-deliveries.ps1`.
