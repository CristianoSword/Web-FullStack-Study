# Compass Workflow

1. Conecte em `mongodb://localhost:27018/customer_crm`.
2. Abra a collection `customers`.
3. Importe o exemplo de `compass/example-document.json` se quiser testar manualmente.
4. Rode os comandos de `queries/crud.mongodb.js` no playground do Compass.
5. Confirme o estado final com `queries/verification.mongodb.js`.
