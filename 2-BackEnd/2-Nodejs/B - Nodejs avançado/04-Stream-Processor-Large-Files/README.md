# 04-Stream-Processor-Large-Files

Mini laboratorio para praticar agregacao de dados inspirada em processamento por streams.

## Arquivos

- `src/services/report-processor.js`: soma linhas e montantes.
- `src/cli/run-report.js`: ponto de chamada do fluxo.
- `src/validators/report-validator.js`: descarta linhas mal formadas.
