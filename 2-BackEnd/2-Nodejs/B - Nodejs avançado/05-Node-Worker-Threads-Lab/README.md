# 05-Node-Worker-Threads-Lab

Laboratorio compacto para raciocinar sobre divisao de carga e consolidacao de resultados.

## Arquivos

- `src/models/job-config.js`: define o tamanho do trabalho.
- `src/workers/run-chunk.js`: representa a execucao de um worker.
- `src/services/prime-service.js`: calcula o parcial do intervalo.
- `src/validators/job-validator.js`: protege a configuracao.
