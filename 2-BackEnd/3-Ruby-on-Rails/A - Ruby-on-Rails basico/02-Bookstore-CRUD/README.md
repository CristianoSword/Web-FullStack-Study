# 02-Bookstore-CRUD

Projeto base para estudar CRUD de livros em Rails com rotas REST e validacoes simples.

## Estrutura

- `app/models/book.rb`: modelo com validacoes.
- `app/services/book_catalog.rb`: consultas de listagem.
- `app/controllers/books_controller.rb`: `index` e `create`.
- `config/routes.rb`: recursos de livros.
