# 04-File-Read-Write

Leitura e escrita real de configuracoes em arquivo para estudo de IO simples.

## Arquivos

- `ConfigEntry.java`: chave e valor.
- `FileConfigService.java`: cria, escreve e le o arquivo `data/app.properties`.
- `Main.java`: imprime o caminho do arquivo e a primeira configuracao carregada.
