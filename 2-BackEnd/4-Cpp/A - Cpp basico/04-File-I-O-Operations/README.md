# 04-File-I-O-Operations

Leitura real de registros em arquivo CSV com agregacao simples de estatisticas.

## Arquivos

- `include/record.hpp`: estrutura de entrada.
- `include/report_reader.hpp`: leitura de arquivo e agregacao.
- `data/records.csv`: base inicial para o laboratorio.
- `src/main.cpp`: exibicao do resumo final.
