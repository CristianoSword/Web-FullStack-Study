Acordeão Nativo (Details)

O que é HTML5?

HTML5 é a quinta e última versão principal do HTML, trazendo APIs para áudio, vídeo, canvas e muito mais.

Precisa de JavaScript?

Não! O comportamento de abrir e fechar é 100% nativo do navegador usando o atributo booleano [open].

* Nota: Animar a altura do details de forma nativa ainda é um desafio, mas o estado pode ser estilizado facilmente.
