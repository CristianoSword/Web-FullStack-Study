# 🎯 Módulo jQuery Basics

Este projeto demonstra os fundamentos essenciais da biblioteca jQuery através de um Gerenciador de Tarefas dinâmico.

## 🛠️ Fundamentos Aplicados:
1. **Seletores**: ID (`#`), Classe (`.`) e Tags.
2. **Eventos**: Clique (`click`), Teclado (`keypress`) e Delegação (`on`).
3. **DOM**: Adição (`append`), Remoção (`remove`) e Valores (`val`).
4. **CSS**: Toggle de classes (`toggleClass`) e manipulação de estilos.
5. **Efeitos**: Transições de entrada e saída (`fadeIn`, `fadeOut`).
6. **Traversal**: Acesso a elementos pai (`parent`) e filtros.
