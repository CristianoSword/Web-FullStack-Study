# 🌤️ Weather Dashboard Pro

Um dashboard de clima completo e profissional desenvolvido com **jQuery Avançado**, focado em performance, UX e consumo de APIs assíncronas.

## 🚀 Tecnologias e Conceitos:
1. **jQuery Advanced**: Manipulação complexa de DOM, delegação de eventos e animações encadeadas.
2. **AJAX (Asynchronous JavaScript and XML)**: Consumo da API OpenWeather com tratamento de Promises e interceptação de erros.
3. **Geolocation API**: Detecção automática da localização do usuário.
4. **LocalStorage**: Persistência de preferências (unidades C/F) e histórico de buscas.
5. **Responsive Design**: Interface adaptável para Mobile, Tablet e Desktop.
6. **Timezone Handling**: Cálculo de hora local baseado no fuso horário da cidade buscada.
7. **Dynamic UI**: Backgrounds e ícones que mudam conforme a condição climática.

## 🛠️ Como usar:
1. Obtenha uma chave de API em [OpenWeatherMap](https://openweathermap.org/api).
2. Substitua o valor de `API_KEY` no arquivo `src/services.js`.
3. Abra o `index.html` em seu navegador.
