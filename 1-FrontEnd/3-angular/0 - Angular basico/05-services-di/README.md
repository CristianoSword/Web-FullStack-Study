# 05 - Serviços e Injeção de Dependência

Este projeto demonstra o uso de Serviços para gerenciar estado e lógica de negócio de forma desacoplada dos componentes.

## Conceitos Aplicados
- `@Injectable()`: Definindo um serviço.
- `providedIn: 'root'`: Singleton service.
- `inject()` function: Nova forma de injeção de dependência no Angular (v14+).
- Separação de responsabilidades (Data vs UI).

## Como rodar
1. `npm install`
2. `npm start`
