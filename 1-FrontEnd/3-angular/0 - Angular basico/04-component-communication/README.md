# 04 - Comunicação entre Componentes

Este projeto explora como os componentes se comunicam no Angular usando decorators @Input e @Output.

## Conceitos Aplicados
- `@Input()`: Receber dados do pai.
- `@Output()` + `EventEmitter`: Enviar eventos para o pai.
- Standalone components aninhados.
- Passagem de dados estáticos e tratamento de eventos.

## Como rodar
1. `npm install`
2. `npm start`
