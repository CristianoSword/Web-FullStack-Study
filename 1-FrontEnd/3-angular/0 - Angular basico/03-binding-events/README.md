# 03 - Property & Event Binding

Este projeto demonstra como o Angular lida com dados fluindo do componente para o template (Property Binding) e do template para o componente (Event Binding).

## Conceitos Aplicados
- Property Binding (`[property]="value"`)
- Event Binding (`(event)="handler()"`)
- Template Reference Variables (`#variable`)
- Style Binding (`[style.backgroundColor]`)

## Como rodar
1. `npm install`
2. `npm start`
