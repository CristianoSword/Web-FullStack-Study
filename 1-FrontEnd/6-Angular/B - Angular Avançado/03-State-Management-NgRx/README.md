# ⚡ 03-State-Management-NgRx — AuraNgRx

Aplicação de vitrine gamer e carrinho de compras reativo utilizando a arquitetura oficial de **Gerenciamento de Estado Global do NgRx** no **Angular 18**.

---

## 🎨 Principais Recursos & UX

1. **NgRx Redux Architecture**: Fluxo unidirecional completo com **Actions**, **Reducers** e **Selectors** fortemente tipados.
2. **Dynamic Shopping Drawer**: Carrinho flutuante em painel lateral responsivo que monitora dinamicamente a contagem e valores do estado global.
3. **Calculador de Total Reativo**: Uso de Selectors compostos para calcular o valor acumulado em tempo real com injeção automática no HTML via `async` pipe.
4. **Seed Gamer Showcase**: Vitrine premium de periféricos e hardwares de alta performance gamer com controle individual de adições.

---

## 🛠️ Tecnologias Utilizadas

* **Angular 18**: Componentes autônomos e fluxo de controle reativo (`@for`, `@if`).
* **NgRx Store v18**: Gerenciamento de estado com `provideStore`.
* **Design Premium**: Estilo cyberpunk com paleta esverdeada de altíssima fidelidade.

---

## 🚀 Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
3. Abra o navegador em: **`http://localhost:4200`**
