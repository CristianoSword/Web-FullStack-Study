# 🎨 04-Custom-Component-Library — AuraUI

Biblioteca corporativa de componentes de UI reusáveis, modulares e altamente customizáveis construídos sobre a arquitetura autônoma (Standalone) do **Angular 18**.

---

## 🎨 Principais Recursos & UX

1. **AuraRippleDirective**: Diretiva customizada que escuta cliques no DOM para calcular dimensões e injetar de forma animada um efeito tátil de ripple neon.
2. **AuraButtonComponent**: Botão parametrizado por variantes (`primary`, `secondary`, `outline`, `danger`) com estilizações HSL de altíssima fidelidade.
3. **AuraBadgeComponent**: Tags sinalizadoras de status e alertas com bordas semitransparentes e glow.
4. **AuraCardComponent**: Recipientes modulares que utilizam **Content Projection (`<ng-content>`)** para projeção flexível de conteúdo HTML e controle de elevações de hover.

---

## 🛠️ Tecnologias Utilizadas

* **Angular 18**: Componentização standalone avançada e encapsulamento de estilos.
* **Directives & ElementRef**: Manipulação de DOM segura com `Renderer2`.
* **Design Premium**: Estilo cyberpunk violeta orquídea com micro-animações táteis.

---

## 🚀 Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
3. Abra o navegador em: **`http://localhost:4200`**
