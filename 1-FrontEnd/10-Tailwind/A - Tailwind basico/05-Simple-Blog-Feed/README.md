# 📦 05 – Simple Blog Feed

Feed de notícias com cards estruturados, categorias coloridas e efeitos de transição usando **Tailwind CSS**.

## 🚀 Como executar

Abra `index.html` diretamente no navegador (usa CDN do Tailwind).

## 📚 Conceitos
- `group` e `group-hover` para efeitos encadeados em elementos filhos
- `hover:shadow-md transition-shadow` para elevação suave
- Layout flexível com `flex gap-6`
- Cores categorizadas: indigo, emerald e rose
