# 📦 03 – Navbar Responsiva

Barra de navegação mobile interativa com toggle usando **Tailwind CSS** e JS básico.

## 🚀 Como executar

```bash
npm install
npm run build
```

Abra `index.html` no navegador. Redimensione para ver o menu mobile.

## 📚 Conceitos
- `hidden md:flex` para esconder/mostrar por breakpoint
- Toggle com JS puro + classe `hidden`
- Transições suaves com `transition-colors`
