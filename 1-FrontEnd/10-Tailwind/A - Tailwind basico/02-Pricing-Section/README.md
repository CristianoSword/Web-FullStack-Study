# 📦 Projeto 02 – Pricing Section

Seção de planos e preços responsiva com hover states modernos, construída com **Tailwind CSS**.

## 🚀 Como executar

```bash
npm install
npm run build   # compila o CSS
npm run watch   # modo watch
```

Abra `index.html` no navegador.

## 📚 Conceitos

- Grid responsivo com `md:grid-cols-3`
- Hover states com `hover:shadow-xl` e `transition`
- Destaque de plano popular com posicionamento absoluto
- Paleta de cores coesa com `indigo` como cor principal
