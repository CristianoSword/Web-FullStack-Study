# 📦 04 – Form Validation UI

Formulário de cadastro com validação visual de estados de erro e sucesso usando **Tailwind CSS** e JavaScript nativo.

## 🚀 Como executar

Abra `index.html` diretamente no navegador (usa CDN do Tailwind).

## 📚 Conceitos
- Estados visuais de erro com `border-red-400`
- Feedback de sucesso com `bg-green-50`
- Validação de e-mail com regex
- `focus:ring-2` para acessibilidade
