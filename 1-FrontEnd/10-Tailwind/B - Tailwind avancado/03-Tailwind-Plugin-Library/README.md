# 03 - Tailwind Plugin Library

Biblioteca demonstrando plugins customizados do Tailwind para criar componentes e utilities reutilizaveis.

## Como executar

Abra `index.html` diretamente no navegador.

## Conceitos

- `tailwind.config.plugins` com `addComponents` e `addUtilities`
- Componentes reutilizaveis como `.surface-card`, `.button-solid` e `.status-pill`
- Layout responsivo com utility customizada `.content-grid`
- Preview visual para validar a API da biblioteca
