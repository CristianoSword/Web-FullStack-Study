# 📦 01 – Tailwind Theme Customizer (Avançado)

Dashboard com alternância de tema dark/light e seletor de cor de accent em tempo real usando **Tailwind CSS**.

## 🚀 Como executar

Abra `index.html` diretamente no navegador.

## 📚 Conceitos
- `darkMode: 'class'` e toggle via JavaScript
- `transition-colors duration-300` para transições suaves
- Seletor de accent color dinâmico
- Configuração inline do `tailwind.config`
