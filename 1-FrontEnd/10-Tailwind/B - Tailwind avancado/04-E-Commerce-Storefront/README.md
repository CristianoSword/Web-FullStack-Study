# 04 - E-Commerce Storefront

Catalogo de loja com filtros por categoria, busca local, ordenacao de preco, cards responsivos e carrinho simulado.

## Como executar

Abra `index.html` diretamente no navegador.

## Conceitos

- Layout responsivo com sidebar de filtros
- Renderizacao dinamica de produtos com JavaScript
- Busca, ordenacao e filtro por categoria
- Transicoes e estados visuais usando classes utilitarias do Tailwind
