# Todo App - Vue 3 Composition API

Este é o projeto 01 do módulo Vue.js Básico do repositório **Web-FullStack-Study**.

## Conceitos Abordados
* **Composition API**: Uso de `<script setup>` com `ref` e `computed`.
* **Reatividade Direta (`v-model`)**: Vínculo bidirecional no input de nova tarefa.
* **Diretivas do Vue**: `v-for` com chaves únicas, `v-if`, `v-else`, `@submit.prevent` e `@click`.
* **Propriedades Computadas (`computed`)**: Estatísticas em tempo real e lista filtrada.
* **Lifecycle Hooks (`onMounted`)**: Carregamento assíncrono do localStorage.
* **Efeitos de Transição**: Utilização do componente nativo `<TransitionGroup>`.
* **Estilização Premium**: Visual moderno e responsivo com efeito de Glassmorphism.
