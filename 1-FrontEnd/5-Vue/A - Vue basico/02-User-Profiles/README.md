# User Profiles - Vue 3 Consumo de API

Este é o projeto 02 do módulo Vue.js Básico do repositório **Web-FullStack-Study**.

## Conceitos Abordados
* **Ciclos de Vida (`onMounted`)**: Disparo de carregamento inicial dos dados assim que o componente é inserido no DOM.
* **Operações Assíncronas (Async/Await & Fetch)**: Integração com a API REST pública `randomuser.me`.
* **Estados de Interface**: Controle reativo de estados de carregamento (`isLoading`), dados obtidos e cenários de erro (`error`).
* **Busca Computada**: Filtragem reativa baseada em strings em tempo de execução via `computed`.
* **Esqueleto Shimmer**: Animações de esqueleto de carregamento personalizadas em CSS puro.
