# 02 - Quiz-Master (Web Quiz)

Uma engine de quiz interativa rodando no browser, focada em tipagem de estado e manipulao segura de DOM.

## Conceitos Aplicados
- **Union & Literal Types**: Definio de dificuldades e estados do quiz.
- **Type Casting**: Manipulao de elementos do DOM com `as HTMLButtonElement`.
- **Modificadores de Acesso**: Proteo do estado interno da engine.
- **Vite + TypeScript**: Setup moderno de desenvolvimento web.

## Como Executar
1. Instale as dependncias:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Abra a URL exibida no terminal.
