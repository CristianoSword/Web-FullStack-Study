# 04 - WeatherHub (API Dashboard)

Um dashboard de clima que demonstra o consumo de APIs com tipagem rigorosa e uso de caches.

## Conceitos Aplicados
- **Async/Await com Tipagem**: Promessas tipadas para chamadas assncronas.
- **Type Guards**: Funo `isWeatherData` para validar objetos em tempo de execuo.
- **Maps**: Uso da estrutura `Map` do TS para implementar um sistema de cache com expirao.
- **Enums**: Padronizao de condies climticas.

## Como Executar
1. Instale as dependncias:
   ```bash
   npm install
   ```
2. Inicie o servidor:
   ```bash
   npm run dev
   ```
