# 02 - MiniInversify (DI Framework)

Um mini-framework de Injeo de Dependncia inspirado no InversifyJS e NestJS, utilizando decoradores e metadados.

## Conceitos Avanados Aplicados
- **Decorators**: Uso de `@Injectable()` e `@Inject()` para metaprogramao.
- **Reflect Metadata**: Extrao de tipos de parmetros de construtor em tempo de execuo.
- **Singleton Pattern**: Gesto de instncias nicas dentro de um container.
- **Dependency Resolution**: Algoritmo de resoluo recursiva com deteco de dependncia circular.

## Como Executar
1. Instale as dependncias:
   ```bash
   npm install
   ```
2. Execute o demo:
   ```bash
   npm run dev
   ```
