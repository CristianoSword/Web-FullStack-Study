# 04 - TypedSDK (HTTP Client)

Um cliente HTTP (SDK) universal e fortemente tipado que gera mtodos de forma dinmica baseando-se em uma definio de API.

## Conceitos Avanados Aplicados
- **Recursive Template Literals**: Extrao de parmetros de rota (ex: `:id`) diretamente de strings de path.
- **Advanced Overloads**: Uso de rest parameters e conditional types para tornar argumentos obrigatrios ou opcionais com base no endpoint.
- **Interceptors**: Padro de design para processamento de requisies em cadeia.
- **Union to Intersection**: Transformao de tipos complexos para garantir compatibilidade total.

## Como Executar
1. Instale as dependncias:
   ```bash
   npm install
   ```
2. Execute o demo:
   ```bash
   npm run dev
   ```
