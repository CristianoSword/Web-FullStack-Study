# ⚡ 05-Custom-Plugin-Integration — AuraAnalytics

Console analítico integrado com o plugin de gráficos de terceiros **Chart.js** acoplado de forma fluida a um layout responsivo baseado em **Bootstrap 5**.

## 🚀 Recursos
- **Chart.js Engine**: Instanciação de gráficos vetoriais complexos configurados sob medida para o tema escuro.
- **Dataset Update Simulation**: Lógica no cliente para atualizar dinamicamente todos os pontos de dados da série de largura de banda e de CPU com transições nativas.
- **Audit Logs Table**: Tabela de sensores e logs de telemetria perfeitamente alinhada em dispositivos móveis por meio de contêineres `.table-responsive`.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Grids, custom Badges, responsive Tables)
- Chart.js (External Vector Graphics Library)
- CSS3 (Plus Jakarta Sans Font & Glassmorphic themes)
- Vanilla JavaScript
