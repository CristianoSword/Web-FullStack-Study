# ⚡ 03-SaaS-Pricing-Wizard — Planos Interativos

Matriz comparativa de precificação SaaS interativa baseada em **Bootstrap 5** com chaveamento reativo de faturamento mensal e anual.

## 🚀 Recursos
- **Billing Switcher**: Alternância dinâmica de preços em tempo real com indicador de economia de 20% no modo anual.
- **Popular Card Accent**: Destaque visual com borda neon e badge personalizado para o plano recomendado.
- **FAQ Accordion**: Seção de suporte operacional estruturada com componentes Accordion nativos do Bootstrap.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Grids, Forms Switch, custom Badges, Accordions)
- CSS3 (Plus Jakarta Sans Font & Glow-hover effects)
- Vanilla JavaScript
