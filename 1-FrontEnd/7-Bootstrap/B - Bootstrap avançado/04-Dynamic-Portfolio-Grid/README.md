# ⚡ 04-Dynamic-Portfolio-Grid — Engenharia Showcase

Grid interativo de portfólio de engenharia de software baseado em **Bootstrap 5** com transições de zoom dinâmico e filtragem por tags no cliente.

## 🚀 Recursos
- **Interactive Tags Filter**: Filtragem de projetos instantânea utilizando listeners e readequação de layouts.
- **Micro-interações premium**: Ampliação e foco de cartões com transições suaves.
- **Fade-In Keyframe**: Animação personalizada `@keyframes fadeIn` ativada nativamente ao remontar os contêineres na viewport.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Grids, custom Buttons, responsive Cards)
- CSS3 (Plus Jakarta Sans Font, scaling & fade-in animations)
- Vanilla JavaScript
