# ⚡ 02-Dashboard-Pro-Bootstrap — Telemetry Console

Console analítico de telemetria avançado baseado em **Bootstrap 5** com transições táteis, animações de pulso e relatórios operacionais em tempo real.

## 🚀 Recursos
- **Simulador de Telemetria**: Algoritmo reativo no cliente que atualiza a latência e a taxa de erro com micro-animações de pulso visual.
- **Widgets Customizados**: Cartões HSL com variações de badges informativas de crescimento.
- **Process List Grid**: Tabela de processos encapsulada em `.table-responsive` para visualização estável em dispositivos mobile.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Grids, custom Badges, Tables)
- CSS3 (Plus Jakarta Sans Font, glow-pulse animations)
- Vanilla JavaScript
