# ⚡ 01-Custom-Themed-Bootstrap — SASS Customized Theme

Showcase avançado contendo a customização profunda do **Bootstrap 5** através da sobrescrita de variáveis fundamentais em SASS compiles.

## 🚀 Recursos
- **SASS Integration**: Estrutura pronta para compilação local de arquivos `.scss` para CSS final.
- **Color Swatches Grid**: Painel operacional exibindo as amostras de cores correspondentes a `$primary` (orquídea) e `$success`/`$info`.
- **Dynamic Brightness Control**: Lógica no cliente para alternar níveis de luminância de fundo na viewport.

## 🛠️ Compilação local SASS
Execute o comando abaixo na pasta raiz para recompilar alterações nas variáveis:
```bash
npm run build:sass
```

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (SASS Source code Integration)
- SCSS / SASS variables override
- Vanilla JavaScript
