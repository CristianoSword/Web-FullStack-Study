# ⚡ 05-Interactive-Form-UI — AuraAccess

Interface responsiva de formulário seguro contendo validação de campos estrita baseada nas classes nativas de feedback visual do **Bootstrap 5** (como `.was-validated` e `.invalid-feedback`).

## 🚀 Recursos
- **Validação Nativa HTML5**: Bloqueio de submissões de dados vazios ou incompatíveis no cliente.
- **Micro-interações de Foco**: Efeito estético de glow neon magenta ao selecionar os inputs.
- **Checkbox com Gatilho**: Caixa de seleção obrigatória de aceitação dos termos com feedback de erro dinâmico.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Forms, Input Groups, validation Feedbacks)
- CSS3 (Plus Jakarta Sans Font & Magenta Neon styles)
- Vanilla JavaScript
