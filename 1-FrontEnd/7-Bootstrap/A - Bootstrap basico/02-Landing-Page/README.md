# ⚡ 02-Landing-Page — AuraCore

Página institucional totalmente responsiva e esteticamente premium construída sobre a biblioteca **Bootstrap 5** com refinamentos manuais HSL e efeitos translúcidos (glassmorphic).

## 🚀 Recursos
- **Sistema de Grid Fluido**: Alinhamento adaptável de seções de herói, recursos e depoimentos.
- **Glassmorphism Header**: Barra de navegação semitransparente com efeito `backdrop-filter` para foco no conteúdo.
- **Aura Glow Effects**: Orbes fluorescentes estilizadas geradas nativamente por gradientes radiais.
- **Data Rendering**: Mapeamento estruturado das seções carregadas via módulos de JavaScript puro.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (CSS/JS)
- CSS3 Custom Properties (Tema Escuro Cyberpunk)
- Vanilla JavaScript
