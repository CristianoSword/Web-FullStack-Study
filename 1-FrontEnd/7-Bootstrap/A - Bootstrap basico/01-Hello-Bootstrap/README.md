# ⚡ 01-Hello-Bootstrap — Component Showcase

Showcase operacional interativo apresentando os componentes e as ferramentas básicas do **Bootstrap 5** com paleta de cor violeta neon e responsividade adaptável.

## 🚀 Recursos
- **Showcase Cards**: Visualizadores estáticos de componentes de botões, alertas, spinners e cartões.
- **Painel Interativo de Ação**: Botões programados com JavaScript puro para controlar spinners e simular avisos no cliente.
- **Accordion de Diretrizes**: Perguntas frequentes construídas com o componente Accordion do Bootstrap.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Accordions, Spinners, Buttons, Alerts)
- CSS3 (Plus Jakarta Sans Font & Theme Colors)
- Vanilla JavaScript
