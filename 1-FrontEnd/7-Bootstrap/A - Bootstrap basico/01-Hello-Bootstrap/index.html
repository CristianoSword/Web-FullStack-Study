📚 Diretrizes do Grid System

O Bootstrap divide horizontalmente o layout em 12 colunas virtuais. Usando classes como `.col-md-6` ou `.col-lg-4`, definimos o tamanho proporcional de cada componente nos diferentes breakpoints de tela.

Breakpoints representam faixas de resolução de tela (como `sm`, `md`, `lg`, `xl`). O Bootstrap usa media queries baseadas em largura de tela (mobile-first) para alternar as regras visuais automaticamente.
