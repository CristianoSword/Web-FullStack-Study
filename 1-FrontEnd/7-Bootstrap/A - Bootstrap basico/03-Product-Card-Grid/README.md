# ⚡ 03-Product-Card-Grid — AuraGear

Catálogo de produtos interativo com sistema de filtragem dinâmica por categoria responsiva usando **Bootstrap 5** utilitários nativos e refinamentos estéticos cyberpunk.

## 🚀 Recursos
- **Filtros por Categoria**: Alternância reativa de produtos em tempo de execução sem recarregar a página.
- **tactile Hover Interactions**: Efeito de zoom suave de 1.08x na imagem e alteração de bordas neon no card.
- **Compra e Notificações**: Gatilho reativo de confirmação com alertas integrados.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Grids & Badges)
- CSS3 (Space Grotesk Font & Glow Neon effects)
- Vanilla JavaScript
