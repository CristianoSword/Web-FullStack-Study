# ⚡ 04-Admin-Layout-Base — AuraConsole

Layout estrutural básico para painéis administrativos responsivos contendo barra lateral colapsável dinâmica baseada em **Bootstrap 5** e Vanilla JS.

## 🚀 Recursos
- **Collapsible Sidebar**: Alternância de visibilidade da barra de navegação com ajustes automáticos na viewport principal.
- **Grids e Tabelas Responsivas**: Telemetrias e relatórios protegidos por contêineres `.table-responsive` que impedem overflow lateral em smartphones.
- **Micro-interações**: Feedback visual de foco nos links do menu administrativo.

## 🛠️ Tecnologias
- HTML5
- Bootstrap 5 (Grids, Navbar, responsive Tables, Sidebar)
- CSS3 (Plus Jakarta Sans Font & Theme Colors)
- Vanilla JavaScript
