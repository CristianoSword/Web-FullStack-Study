# 06 - Pixel Art Editor

Editor avancado de pixel art em Svelte com ferramentas de desenho, paleta customizavel, grid ajustavel, preview em canvas, undo/redo e exportacao PNG.

## Como executar

```bash
npm install
npm run dev
```

## Funcionalidades

- Pencil, eraser, fill e picker
- Canvas 16x16, 24x24, 32x32 e 48x48
- Zoom visual do grid sem alterar a arte
- Historico com undo/redo
- Exportacao em PNG com escala pixel-perfect
