Less-Sass avancado 03

Design tokens compilados de JSON para Sass

O script converte `tokens/tokens.json` em `_tokens.scss`, e o Sass usa esses valores para gerar esta interface.

Fonte unica de verdade

Altere o JSON, rode o compilador e todo o tema acompanha os tokens.

color.primary#3157ff
color.secondary#08a88a
space.xl2rem
radius.lg14px
Fluxo completo

Compile tokens e CSS em um comando.

npm run build:sass
