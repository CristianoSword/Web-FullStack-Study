# Less Buttons

Projeto basico para praticar componentes de botao com **Less**.

## O que o exemplo mostra

- variaveis para cores, fonte, sombras e espacamento;
- mixins para gerar estilos reutilizaveis;
- variantes `solid`, `outline` e `ghost`;
- tamanhos `small`, `default` e `large`;
- estados `hover`, `focus-visible`, `active` e `disabled`.

## Como usar

```powershell
npm install
npm run build:less
```

Depois abra `index.html` no navegador.
