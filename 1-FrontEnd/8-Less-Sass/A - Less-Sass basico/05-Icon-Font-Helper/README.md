# ⚡ Projeto 05: Icon-Font Helper & Utility Generator

Este projeto constrói um gerador de classes utilitárias para ícones em lote (tamanhos, cores e efeitos neon/glow) utilizando loops Sass a partir de mapas de configuração.

## 🚀 Conceitos Demonstrados

* **Geração em Lote via `@each`:** Reduz drasticamente a repetição de código CSS gerando centenas de combinações de tamanhos (`icon-size--`) e cores (`icon-color--`) automaticamente.
* **Efeitos Adicionais combinados:** Criação de classes auxiliares complexas (como `.icon-glow`) integradas no mesmo ciclo de iteração do loop.

## 📦 Como Executar

```bash
npm install
npm run build:sass
```
Abra o `index.html` no navegador.
