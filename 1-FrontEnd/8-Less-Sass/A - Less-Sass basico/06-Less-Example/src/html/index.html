Less basico 06

Variaveis, mixins e nesting trabalhando juntos

Um exemplo visual pequeno para mostrar como Less organiza estilos reutilizaveis sem transformar o CSS em repeticao manual.

Ver componentes Ver conceitos

Variaveis

Cores, fontes, raios e espacamentos saem de tokens no `_variables.less`.

Mixins

Sombras, botoes e paineis usam blocos reaproveitaveis com parametros.

Nesting

Estados e elementos internos ficam perto do componente que eles estilizam.

Fluxo do projeto

Edite Less, compile CSS

O comando abaixo gera `src/css/style.css`, que e carregado por esta pagina.

npm run build:less
